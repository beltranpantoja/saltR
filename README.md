# saltr <a href="https://beltranpantoja.github.io/saltR/"><img src="man/figures/saltr_logo.png" align="left" height="138" style="height: 138px"/></a>

The goal of saltR, pronounced as the spanish word *saltar*, is to
simplify the running of simulations in the context of DCMs.

## Installation

You can install the development version of saltr from
[GitHub](https://github.com/beltranpantoja/saltR) with:

``` r
devtools::install_github("beltranpantoja/saltR")
```
