# Get the matrix of the possible ideal responses patterns from a qmatrix

Get the matrix of the possible ideal responses patterns from a qmatrix

## Usage

``` r
get_ideal_patterns(qmat, rule)
```

## Arguments

- qmat:

  Q-matrix from which to generate the patterns

- rule:

  DINA (only students with all relevant attributes should have the item
  correct) or DINO (students with at least one relevant attribute should
  have the item correct)

## Value

a matrix of responses patterns and items
